Docker and Docker Compose:

docker === open-source containarization platform. It enables developers to package applications into containers - standardized executable components combining
application source code with OS libraries and dependencies required to run that code in any environment
docker compose === run multiple containers as a service from 1.yaml file
docker build  === creates new branch
docker start container_id === starts existing container
docker stop container_id === stops existing container
docker run tag_name === creates new container and runs it
docker ps === shows active docker containers
docker ps --all === shows active and not active containers
docker-compose up === build and run project
docker-compose down === stop containers and remove containers
